JavaScriptComponent - это простой компонент пользовательского интерфейса, который представляет собой универсальную обёртку для использования любой готовой библиотеки JavaScript в приложении на базе Jmix напрямую, без необходимости подключать компонент Vaadin.

В этом примере мы будем взаимодействовать с JavaScript API для GoogleMaps. С помощью клика по карте можно определить текущие координаты, а с помощью кнопки Добавить можно добавить маркер по указанным координатам.

Для того, чтобы иметь доступ к GoogleMaps вам обязательно нужен ключ.

jsComponent имеет следующие зависимости:

  • файл коннектора, расположенный в директории resources/VAADIN/googlemaps;
  • файл со стилями, расположенный в директории resources/VAADIN/googlemaps.
