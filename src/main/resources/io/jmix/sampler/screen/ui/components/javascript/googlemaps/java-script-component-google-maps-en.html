JavaScriptComponent is a simple UI component that can work with any JavaScript wrapper without Vaadin component implementation. Thus, you can easily integrate any pure JavaScript component in your Jmix-based application.

In this sample we will be interacting with GoogleMaps JavaScript API. By clicking on the map, you can determine the current coordinates, and using the Add button, you can add a marker at the specified coordinates.

In order to have access to the GoogleMaps, you definitely need an API key.

jsComponent has the following dependencies:

  • connector is placed in the resources/VAADIN/googlemaps directory;
  • style is placed in the resources/VAADIN/googlemaps directory.
